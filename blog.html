FAQ

1)What is DOM and what is the purpose of DOM?

Answer:DOM- Document Object Model which represents Html document like an tree-like structure of a object. The purpose of the DOM is to provide us a way to access and manipulate the content of web pages which makes website interactive.


2)How will you select HTML elements using DOM? Name the DOM methods.

Answer:We can use various methods and properties to select HTML elements using the DOM. The methods to select elements of the html are- getElementById , getElementsByTagName , getElementsByClassName , querySelector , querySelectorAll


3)What is event bubble?

Answer:Event Bubbling happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.


4) What is a callback function and why will you use it?

Answer:A callback is a function passed into another function as a parameter to be executed later.
In programming, we use callback functions to handle user input, network requests, and such kind of events.
